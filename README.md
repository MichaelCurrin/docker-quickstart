## Installation

Follow the [Download and install](https://docs.docker.com/get-docker/) guide on the Docker docs site.

Then clone the repo.

```sh
$ git clone git@github.com:MichaelCurrin/docker-quickstart.git
$ cd docker-quickstart
```
